Incremental backups (CBT)

Uses VMware Changed Block Tracking to store deltas after an initial full backup. Requires local or NFS repository (not S3).

When off, every run is a full VM copy to a date folder.

Full backup interval After this many incremental backups, VMExec creates a synthetic full and trims the chain.

S3 storage is selected — incremental backups are not available. VMExec will use full backups only.
